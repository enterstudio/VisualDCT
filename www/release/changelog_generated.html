v2.1 ms build 1226, released 2001-12-09
fix Not all fields are available as link destinations
comment Need a better plan? Suggestion: Define field order in DBDs prioritized by usage.
fix Print Preview doesn't erase previous preview.
fix Links having process options.
enhancement Pan when the cursor hits the edge of the drawing (while dragging).
enhancement Usage of JAXP 1.1 compilant Apache Crimson XML parser.
enhancement All setting files are now stored in VDCT_DIR/config directory, where VDCT_DIR is environment variable (current directory if not specified or invalid).
enhancement Plugins (alpha state - currently DebugPlugin and ExportPlugin are supported). Configuration file is config/plugins.xml.
comment The "One Up Level" button in javax.swing.FileChooser is part of JRE implementation. No problems noticed on Windows platform.
comment For proper font scaling fonts have to be "flexible" (bitmap fonts are not recommended). No problems noticed on Windows, Linux (XFree86 4.0.3) platform.
bug Process options shown by link fields are not properly centered, they can be overdrawn by a link line (large zoom in).
comment VisualDCT does not handle invalid DBs having unquoted substitution variables as record names (see EPICS IOC Application Developers Guide, R3.13, June 18, page 35).
comment Should connectors be drawn always over other objects? Anyway connectors can be easily accessed (hilited, clicked, dragged) even if they are under other object.
